# Keyboard Shortcuts

Shortcuts are active when no modal is open and focus is not in a text input.

| Key | Action |
|-----|--------|
| **N** | Open new trade modal |
| **P** | Open positions modal |
| **S** | Open settings modal |
| **H** | Open help / welcome guide |
| **D** | Toggle dark / light mode |
| **Esc** | Close any open modal |
